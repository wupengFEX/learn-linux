# cat

`cat` 命令连接文件并打印到标准输出设备上，`cat` 经常用来显示文件的内容。

## 语法

`cat(选项)(参数)`

## 参数

- `-n`: 有 1 开始对所有输出的行数编号；
- `-b`: 和 `-n` 相似，只不过对于空白行不编号；
- `-s`: 当遇到有连续两行以上的空白行，就代换为一行的空白行；
